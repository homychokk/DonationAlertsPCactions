# DonationAlerts PC Actions

Мощный локальный сервер на Node.js, который позволяет управлять вашим ПК во время стрима за донаты через DonationAlerts. Когда зритель отправляет донат, программа может нажимать кнопки на клавиатуре, менять сенсу мыши, инвертировать мышь, запускать команды на ПК или открывать видео на YouTube.

## Что умеет программа:
* **Нажатие клавиш (`press_keys`)**: Нажатие любых клавиш (например, `F` или `4`) при получении доната. Поддерживает сочетания клавиш через синтаксис SendKeys (например, `{F7}disconnect{ENTER}`).
* **Бинды с задержкой (`press_binds`)**: Нажимает указанную кнопку, ждет N секунд (заданную длительность) и нажимает кнопку возврата. Удобно для временного включения/отключения игровой сенсы (через бинды внутри игры).
* **Инверсия мыши (`system_mouse_invert`)**: Настоящая системная инверсия курсора мыши на заданное время. (Использует встроенный компилятор Windows для создания `.exe` "на лету").
* **Скорость мыши (`system_mouse_speed`)**: Системное изменение скорости мыши на определенное время.
* **YouTube (`youtube`)**: Открывает конкретное видео на YouTube или ищет клип по запросу.
* **Streamer.bot и OBS**: Интеграция с OBS WebSocket и вызов действий Streamer.bot.
* **Защита от медиа**: Если зритель прикрепил медиа к донату, системные действия автоматически пропускаются, чтобы не мешать просмотру видео/аудио.
* **Привязка к играм**: Вы можете настроить так, чтобы одни действия работали только когда вы играете в `deadlock`, а другие — в `cs2`.

## 🛠 Установка и Запуск

### Шаг 1: Подготовка
1. Установите [Node.js](https://nodejs.org/ru) (рекомендуется версия LTS).
2. Скачайте этот репозиторий (кнопка `Code` -> `Download ZIP`) и распакуйте в удобную папку.
3. Откройте эту папку, скопируйте файл `config.example.json` и переименуйте копию в `config.json`.

### Шаг 2: Настройка DonationAlerts
1. Зайдите на [DonationAlerts](https://www.donationalerts.com/) под своим аккаунтом.
2. Перейдите в **Настройки -> API и приложения**.
3. Создайте новое приложение:
   * **Название**: Любое (например, `PC Actions`)
   * **Redirect URI**: `http://localhost:3000/auth/callback`
4. После создания вы получите **Client ID** и **Client Secret**.
5. Откройте файл `config.json` в любом текстовом редакторе и вставьте туда ваши данные:
```json
{
  "clientId": "ВАШ_CLIENT_ID",
  "clientSecret": "ВАШ_CLIENT_SECRET",
  ...
}
```

### Шаг 3: Запуск
1. Запустите файл `run.bat` (двойным кликом).
2. При первом запуске откроется браузер. Вас попросят авторизовать приложение в вашем профиле DonationAlerts. Нажмите "Разрешить".
3. После авторизации программа получит токены, сохранит их в `config.json` и начнет слушать входящие донаты!
4. В браузере (по адресу `http://localhost:3000`) доступна панель управления и логи.

## ⚙️ Настройка действий (Actions)

Все триггеры и донатные действия настраиваются в файле `config.json` в массиве `"actions"`. 

### Пример настройки "Нажатие клавиши F за 25 рублей":
```json
{
  "id": "action-2",
  "name": "Нажать F",
  "minAmount": 25,
  "currency": "RUB",
  "type": "press_keys",
  "enabled": true,
  "game": "all",
  "durationMs": 5000,
  "keys": "f"
}
```

### Пример "Инверсия мыши на 10 секунд за 10 рублей":
```json
{
  "id": "action-1",
  "name": "Инверсия мыши",
  "minAmount": 10,
  "currency": "RUB",
  "type": "system_mouse_invert",
  "enabled": true,
  "game": "all",
  "durationMs": 10000
}
```
